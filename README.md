## Installation

To get started with MEV Bot, follow these steps:

1. Clone the repository:
    ```sh
    git clone https://github.com/xxxoney/MEV-by-xxxoney.git
    cd MEV-by-xxxoney
    ```

2. Install the required dependencies:
    ```sh
    pip install -r requirements.txt
    ```

## Usage

1. Run the bot:
    ```sh
    py main.py
    ```
